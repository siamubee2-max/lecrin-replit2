# RECOMMANDATIONS TARIFAIRES

## Position actuelle : SOUS-ÉVALUÉ

Vos prix actuels (9.99€ et 12.99€/mois) sont dans la fourchette basse pour une app de luxe avec des fonctionnalités uniques.

## Recommandations

### Option A : Conserver les prix actuels (Stratégie de pénétration)
- **Avantage** : Acquisition rapide d'utilisateurs
- **Risque** : Perception "pas assez luxe"
- Garder 9.99€ / 12.99€ / 100€ annuel

### Option B : Augmentation modérée (Recommandé)
- **Essentiel** : 14.99€/mois (au lieu de 9.99€)
- **Premium** : 19.99€/mois (au lieu de 12.99€)
- **Annuel** : 149€/an (au lieu de 100€) = 2 mois gratuits
- **Justification** : Alignement avec MILE Club et Drest premium

### Option C : Positionnement luxe affirmé
- **Essentiel** : 19.99€/mois
- **Premium** : 29.99€/mois
- **Annuel** : 249€/an
- **VIP** : 49.99€/mois (nouveau tier avec accès exclusif créateurs)
- **Justification** : Segment luxe, fonctionnalités uniques

## Facteurs à considérer

1. **Apple prend 30%** sur les achats in-app
   - 9.99€ → vous recevez ~7€
   - 19.99€ → vous recevez ~14€

2. **Coût d'acquisition client** dans le luxe est élevé
   - Prix bas = plus de volume mais moins de marge
   - Prix élevé = moins de volume mais meilleure marge

3. **Valeur perçue** dans le luxe
   - Un prix trop bas peut nuire à l'image de marque
   - Les clients luxe s'attendent à payer pour la qualité

## Recommandation finale

**Option B avec ajustement** :
- **Découverte** : GRATUIT (3 essayages/mois) - pour l'acquisition
- **Essentiel** : 14.99€/mois (essayages illimités)
- **Premium** : 24.99€/mois (garde-robe + créateurs)
- **Annuel Premium** : 199€/an (économie de 33%)

Cette structure :
- Offre un tier gratuit pour l'App Store (meilleure conversion)
- Positionne l'app dans le segment premium sans être inaccessible
- Maximise la valeur perçue tout en restant compétitif
